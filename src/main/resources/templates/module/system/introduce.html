    thyme-boot 符合企业级开发基础权限管理系统  2020-01-15

    • 零spring xml配置，简洁的配置，独特的开发体验
    • 提供用户，权限，日志，监控、研发工具等8个功能
    • 防止XSS攻击,通过XssFilter类对所有的输入的非法字符串进行过滤以及替换
    • 控制器层统一的异常拦截机制，@ControllerAdvice统一对异常拦截
    • 前端使用vue,数据双向绑定，使用element ui组件化前端开发方便
    • spring session 共享session放入Redis中利于多集群化部署
    • 完善监控体系，支持去服务器信息监控、driud监控
    • 简化多数据源配置，简化mybatis-plus的配置
